## 🌌 The Autonomous Evolution
The **Autonomous Agentic Tadpole** represents a paradigm shift from "Agent Workspaces" to **Autonomous Sovereign Entities**. Unlike traditional agent platforms, this system operates on a decentralized **Hybrid Actor Model** where the kernel itself is an intelligent, self-correcting participant.

### 🧬 The Sovereign Actor Kernel
The heart of the system is a decoupled Rust kernel that offloads critical I/O and state mutations to isolated background actors:
- **Audit Actor**: A sequential, single-threaded cryptographic sink that ensures the non-repudiation of the Merkle Audit Trail without blocking agent cognition.
- **Memory Actor**: An isolated persistence layer for high-dimensional vector search and relational institutional memory.
- **Autonomous Orchestrator**: A background intelligence loop that scans the environment, evaluates mission health, and dispatches agents to fulfill the **Sovereign Charter**.

## ⚖️ The Aletheia Protocol
All autonomous actions pass through a triple-gate verification loop to ensure safety and alignment with the **Sovereign Governance Charter**:

1.  **Generator**: Proposes a mission or action based on the `AUTONOMY_MANIFEST.md`.
2.  **Verifier**: Scans the proposal against security policies (Budget, Shell Scanner, PII Redactor).
3.  **Reviser**: If verification fails, the kernel triggers a self-annealing loop to fix the proposal before execution.

## 🚀 Quick Start (Autonomous Mode)

### 1. Clone & Initialize
```bash
git clone https://github.com/DDS-Solutions/Autonomous-Agentic-Tadpole.git
cd Autonomous-Agentic-Tadpole
npm install
```

### 2. Seed the Manifest
Define your high-level goals in `directives/AUTONOMY_MANIFEST.md`. The Orchestrator will automatically begin decomposing these into agent tasks.

### 3. Launch the Kernel
```bash
npm run engine
```
Watch the audit logs as the **Orchestrator** begins its first cycle:
`👁️ [Orchestrator] Autonomous loop started. Governance: Aletheia Protocol active.`
