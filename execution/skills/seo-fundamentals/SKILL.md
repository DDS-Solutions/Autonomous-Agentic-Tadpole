# SEO Fundamentals

**Visibility through quality.**

## E-E-A-T
- **Experience**: First-hand.
- **Expertise**: Depth/Credentials.
- **Authoritativeness**: Backlinks/Mentions.
- **Trust**: HTTPS/Accuracy.

## Core Web Vitals
- **LCP** (Load): < 2.5s.
- **INP** (Interactive): < 200ms.
- **CLS** (Stable): < 0.1.

## Technical SEO
- **Structure**: Sitemap, Robots.txt, Canonical, HTTPS.
- **Perf**: Speed, Mobile-friendly, Clean URLs.
- **Schema**: Article, Person, Product, FAQ.

## Content SEO
- **Tags**: Title (50-60c), Desc (150c), H1.
- **Quality**: Depth, Freshness, Uniqueness.

## 🧠 Aletheia Reasoning Protocol (Search Intent)

### 1. Generator (Keyword Intent)
*   **Type**: "Navigational vs Informational vs Transactional?".
*   **User**: "Looking for login or tutorial?".

### 2. Verifier (Quality Check)
*   **Depth**: "Better than top result?".
*   **Speed**: "Bloated images?".
*   **Structure**: "H1 matches intent?".

### 3. Reviser (Optimization)
*   **Meta**: "Clickable description?".
*   **Schema**: "Rich snippets?".

## 🛡️ Security & Safety Protocol (SEO)

1.  **Spam**: No stuffing/hidden text.
2.  **Redirects**: No open redirects.
3.  **XSS**: Sanitize header injections.
4.  **Ethics**: No black hat (cloaking/buying links).
